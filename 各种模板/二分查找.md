# 二分查找

## 二分查找左边界

```cpp
while (l < r) {
    mid = l + r >> 1;
    if (nums[mid] >= target) r = mid;
    else l = mid + 1;
}
```

## 二分查找右边界

```cpp
while (l < r) {
    mid = l + r + 1 >> 1;
    if (nums[mid] <= target) l = mid;
    else r = mid - 1;
}
```

## 二分查找浮点数

```cpp
while (r - l < 1e-8) {
    mid = (l + r) / 2;
    if (mid * mid >= target) r = mid;
    else l = mid;
}
```
